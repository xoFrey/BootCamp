Git Cheat Sheet

Basics commands

git add .

Fügt eine Änderung aus den Arbeitsverzeichnis zu Staging-area hinzu.

git add FILENAME

Hier kann man auch nur eine Datei in die Staging-area hinzufügen.

git commit -m "Text"

Erstellt einen neune commit mit einer Nachricht "Text was geändert wurde"

git push

ladet die dateien die zuvor mit " git add . " in die staging area bewegt worden sind in das Github Repo

git log

hier kann ich mir alle Commits angucken die gemacht worden sind

git status

hier kann ich sehen welche datei veränder worden sind

git pull

Mit "git pull" lade ich die geänderten datein herunter

Branches anlegen und wechseln

git branch

Zeigt alle vorhanden branches an. (* ist wo man sich gerade befindet)

git branch BRANCHNAME

legt einen neune branch an

git checkout BRANCHNAME

wechselt zu einen neuen branch

git checkout -b BRANCHNAME

legt einen neune branch BurgerMenu2 an und wechselt dahin

git branch -d BRANCHNAME

Löscht den branch BRANCHNAME

Ablauf eines Merges

git checkout main

Wechselt zum main branch (Davor muss man sich auf den main branch befinden)

git merge BRANCHNAME
