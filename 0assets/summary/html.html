HTML Zusammenfassung

Verschiedene tags:

  • Links erstellen: a-tag "< a href="#">Home < / a > "
  • Für # kann man den dynamischen oder absoluten Pfad eintragen
  • Dynamischer Pfad:
    • Bsp: ". / summary.html" (Pfad zu einer Datei)
  • Absoluter Pfad:
    • Bsp. "https:www.super-code.de" (URL)
  • h1-h6 tag für Überschriften
    • h1 nur einmal pro Seite verwenden
  • "< nav > < / nav >" - Navigationsleiste
  • p - tag - Für plain text
  • image: < img src="" alt="" > - self closing
    • "alt" steht für alternativer text für Screenreader
  • div - Zum unterteilen, wenn es sinn macht
    • besser mit article, section etc arbeiten
  • < span >< /span > - für Hervorhebungen in Css (nur in kombi mit anderen tags, nie alleine)
  • "< button>Hallo< /button >"
  • "< input placeholder="Halloo" type="password" name="" id="" / >""
  • < label for="Hallo">Hallo Welt< /label >
  • < textarea name="Hallo" id="" cols="30" rows="10"> < /textarea >

Listen

Es gibt zwei Arten von Listen:

Die ordered List:

  1. Befehl "< ol >...< /ol >"
  2. Und dazwischen "< li > ... < /li >"

Und die unordered List:

  • Befehl "< ul >...< /ul >"
  • Und dazwischen "< li > ... < /li >"
  • Zum verschachteln von Listen:
    • "< li > In list item < ul >...< /ul > < /li >"

Block Elemente und Inline Elemente

Beispiele für Block elemente

Nehmen 100% Breite der Seite ein

  • article
  • section
  • div
  • footer
  • header
  • li
  • h1-h6
  • ol und ul
  • p tag

Beispiele für Inline Elemente

Nutzen nur so viel Platz, wie der Content ist

  • a tag
  • Button
  • image
  • input
  • label
  • textarea
  • span
