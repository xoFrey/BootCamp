Box Model

Bechreibt wie sich Block Elemente verhalten.

Der Platzbedarf errechnet sich aus:

  1. width / height (Content-Box)
  2. padding (Padding-Box)
  3. border (Border-Box)
  4. margin (Margin-Box)

Box-sizing property

hilft uns Box Model zu manipulieren

box-sizing: content-box => Default wert, Boxen werden aufaddiert

box-sizing: border-box => Breite wird automatisch mit Border verrechnet.

Margin und Padding

Margin

Ist der Außenabstand

Padding

Ist der Innenabstand

Display property:inline, block, inline-block

inline: Elemente können höhe und breite einnehmen

block: Elemente nehmen 100% der Breite ein

inline-block: Kombiniert das Beste von beiden, man hat nicht immer direkt 100% Breite & man kann Höhe und Breite vergeben.
